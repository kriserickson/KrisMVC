A Mustache implementation in PHP.

http://defunkt.github.com/mustache

Mustache is a framework-agnostic logic-less templating language. It enforces separation of view logic from template files. In fact, it is not even possible to embed logic in the template.

This is very, very rad.

Mustache class constructor.


This method accepts a $template string and a $view object. Optionally, pass an associative  array of partials as well.

			PRAGMA_UNESCAPED
			 =  'UNESCAPED'
			(line 53)
		
	


	

The {{%UNESCAPED}} pragma swaps the meaning of the {{normal}} and {{{unescaped}}}  Mustache tags. That is, once this pragma is activated the {{normal}} tag will not be  escaped while the {{{unescaped}}} tag will be escaped.


Pragmas apply only to the current template. Partials, even those included after the  {{%UNESCAPED}} call, will need their own pragma declaration.
This may be useful in non-HTML Mustache situations.
